# YKS Net Takip - Privacy Policy

This repository hosts the public Privacy Policy page for the YKS Net Takip application.

## About

The page is published as a static website via GitHub Pages and is intended to be used as the official privacy policy URL for store listings and user reference.

## Live URL

After enabling GitHub Pages, your URL will follow this format:

https://USERNAME.github.io/REPOSITORY_NAME/

Example:

https://yourusername.github.io/yks-tracker-policy/

## Project Structure

- index.html: Privacy Policy page content

## Technologies

- HTML5
- Inline CSS

## Local Preview

You can open the project locally by directly launching the HTML file in your browser.

## Deployment

Deployment is handled through GitHub Pages:

1. Push this repository to GitHub.
2. Go to Settings > Pages.
3. Select Deploy from a branch.
4. Choose main branch and /(root) folder.
5. Save and wait for the site to be published.

## Maintenance

To update the policy, edit index.html and push the changes. GitHub Pages will redeploy automatically.
